# SPT Mod Manager

Десктопное приложение для управления модами **Single Player Tarkov (SPT)**.  
Позволяет скачивать моды прямо с [forge.sp-tarkov.com](https://forge.sp-tarkov.com), устанавливать их из архивов и деплоить на удалённый сервер по SSH.

## Возможности

- **Каталог модов** — просматривай, ищи и скачивай моды прямо с Forge не выходя из приложения
- **Умная установка** — автоматически определяет тип мода: клиентский (BepInEx) или серверный (SPT/user)
- **Зависимости** — при скачивании мода автоматически подтягиваются все его зависимости
- **Локальный режим** — копирует серверные моды в папку рядом с игрой
- **SSH режим** — загружает серверные моды напрямую на удалённый сервер
- **Фильтры** — поиск по названию, фильтрация по версии SPT и категории
- **Поддержка форматов** — `.zip`, `.7z`, `.rar`

## Установка

### Готовый установщик

Скачай `SPT Mod Manager Setup x.x.x.exe` из [Releases](../../releases) и запусти.

### Из исходников

```bash
git clone https://github.com/YOUR_USERNAME/SPTManagerMods.git
cd SPTManagerMods
npm install
npm run dev
```

Для сборки установщика:
```bash
npm run build
```

## Начало работы

### 1. Forge API токен (для каталога модов)

Чтобы пользоваться каталогом модов, нужен токен:

1. Зайди на [forge.sp-tarkov.com](https://forge.sp-tarkov.com) и войди в аккаунт
2. Нажми на аватар → **API Tokens**
3. Создай новый токен с разрешением `read`
4. Скопируй токен и вставь в приложении: **Настройки → Forge API токен**

### 2. Путь к игре

В **Настройках** укажи папку с SPT (например `C:\SPT\`).  
Туда будут копироваться клиентские моды (BepInEx) и серверные моды в локальном режиме.

### 3. SSH (опционально)

Если SPT-сервер запущен на отдельной машине:

- Укажи IP, порт, пользователя и пароль (или путь к приватному ключу)
- Укажи путь к папке SPT на сервере (например `/root/SPT/`)
- При установке выбери режим **Удалённо (SSH)**

## Структура модов

| Тип мода | Что ищет в архиве | Куда устанавливает |
|---|---|---|
| Клиентский | Папка `BepInEx/` | `<gamePath>/BepInEx/` |
| Серверный (локально) | Папка `SPT/` | `<gamePath>/SPT/` |
| Серверный (SSH) | Папка `SPT/user/` | `<serverPath>/user/` |

## Стек

- **Electron** — десктопная оболочка
- **React + Vite** — интерфейс
- **adm-zip / node-7z** — распаковка архивов
- **node-ssh** — SSH загрузка
- **fs-extra** — работа с файловой системой
